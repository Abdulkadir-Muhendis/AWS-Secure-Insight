void aws_main_state_machine ( t_aws_kit *  kit)

State machine for Main task.

This task initializes ATECC508A and ATWINC1500 to build certificates, and connect to WIFI AP. The WIFI credential should be saved in slot8 to be able to make connection. If there is no either valid WIFI credential or compressed cert data, LED2 continues to blink.
